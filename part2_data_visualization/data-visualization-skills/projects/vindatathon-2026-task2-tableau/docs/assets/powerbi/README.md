# Power BI Assets

Place manually created Power BI files here if you build the report in Power BI Desktop.

Suggested local file:

```text
VinDatathon_Task2_PowerBI.pbix
```

The project `.gitignore` excludes `.pbix`, `.pbit`, and `.pbip` artifacts by default because they are binary and can become too large for normal GitHub commits. To publish a Power BI file, use Git LFS or attach it to a GitHub Release.

The manual rebuild instructions are in:

```text
docs/powerbi-rebuild-guide.md
```
